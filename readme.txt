When uploading, it will shrink the image automatically. The feature is  keeping the aspect ratio and  maximum file size can be specified.


== Description ==

Image will be shrink in the case of automatic JPEG and PNG format to upload the image. 
There are the following features in isShrinker WordPress Plugin. 

Can specify the maximum file size.
Keep aspect ratio.
Convert image inclination from inclination information of image

If you upload, such as iPhone app slope of image because reduced after correction 
it appears in the blog exactly as it appears in the iPhone. 

The maximum file size of the image is 50KB by default 
Length of the long side of the image is 640px by default 
And it is with, but you can change from the management screen of WordPress.

Upload png image 'watermark.png' to plugin directory and then watermark will be enable.
Watermark add on right bottom of image.
If watermark enabled water mark transparency setting is enable.


== Installation ==

1. Upload `isshrinker.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
